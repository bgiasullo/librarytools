Zooniverse Data Cleanup

This tool accepts a CSV file and performs the following steps:

  1. Removes extraneous text from your Zooniverse data file (things like [{"task":"T4","value":").
  2. Finds and replaces unicode with its corresponding symbol (Example: Replaces \u0026 with &).
  3. Compares multiple annotations for each subject, finds the two most similar, then selects one of these to be the final transcript.
