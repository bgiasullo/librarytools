Split text file into multiple files

Upload a plain text file. The script splits the file on occurrences of Image <number>, removes the marker and a leading Transcription: (if present immediately after the marker), and preserves all line breaks and symbols. Download each split file or a ZIP containing them all.

Or drag & drop a .txt file here
    Notes: This is client-side only — nothing is uploaded to a server. It works in modern browsers. If your file uses another marker (e.g. lowercase image), the script will still match case-insensitively.
